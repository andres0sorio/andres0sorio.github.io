My code repository

CMSWork

This repository contains most of the code that I have developed during my days at the CMS experiment at CERN from 2008 to 2013. It contains all sort of source code for data analysis, emulation of the TTU, and many other experimental code.

GRBsWork

While studying High Energy Neutrinos at The Universidad de los Andes (Uniandes) in closed collaboration with Profesor Sahu, I developed this code that models the oscillation of neutrinos through matter of varying density. Our code was based on the excellent work by Tommy Ohlsson ( for a complete list of his publications visit here ) Thanks to this work, we produced the following two publications using this code.

  • Oscillation of high energy neutrinos in Choked GRBs, Andres Felipe Osorio Oliveros, Sarira Sahu, Juan Carlos Sanabria, Eur.Phys.J. C73 (2013) 2574 DOI 10.1140/epjc/s10052-013-2574-8
  • High energy neutrinos from choked GRBs and their flavor ratio measurement by the IceCube, Karla Varela, Sarira Sahu, Andrés Felipe Osorio Oliveros, Juan Carlos Sanabria, Eur.Phys.J. C75 (2015) 289 DOI 10.1140/epjc/s10052-015-3524-4

DS4A 5.0 Work

Data science for all Colombia course cohort 5. What a great oportunity. I classified among thousands of fellows Colombians to take this course. All the code that I have written, including the code developed to solve the Pre-program projects (all 3), is here. Some code is experimental, some other may be used for our team project.

I created for our team project an Organization. We named our team Route90 and I even created a logo. This way all code related to the project is in there, shared and collaboratively developed between the members of the group. Unfortuanately, due to confidence agreements we all signed on, that code must be Private.
